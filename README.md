# Authentication App

A complete user authentication system with multi-user support and comprehensive form validation.

**Features:**
- SharedPreferences-based multi-user storage (username-prefixed keys)
- Sequential validation with specific error messages
- Password visibility toggle
- Dynamic year spinner (current year to 1900)
- Custom gender input handling
- Age verification (minimum 13 years)
- Back button prevention on authenticated screens

**Tech Stack:** Java, Android SDK, SharedPreferences, Material Design
